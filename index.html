About This Project

This website shows how core networking concepts like DNS, IP addressing, and HTTPS actually work in a real web deployment on GitHub Pages.

Security Features Implemented

  • HTTPS via GitHub Pages — all traffic is encrypted over TLS on port 443
  • Content Security Policy (CSP) — controls which scripts and resources are allowed to load
  • XSS Protection — DOMPurify sanitizes any dynamic user input before it hits the DOM
  • X-Frame-Options: DENY — blocks the site from being embedded in an iframe
  • X-Content-Type-Options: nosniff — stops MIME-type sniffing
  • Referrer-Policy — limits how much referrer data gets sent to third parties

Performance Optimizations

  • Minified CSS — styles are inlined and condensed to cut down on HTTP requests
  • Lazy loading — images only load once they scroll into view
  • Cache-Control headers — assets are cached for 24 hours (86400 seconds)
  • Automated deployment — GitHub Actions CI/CD pipeline runs on every push
