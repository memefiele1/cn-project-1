2. IP Addressing

This site resolves to both IPv4 and IPv6 addresses. The IPv4 address 185.199.110.153 is a 32-bit address in dotted-decimal format. The IPv6 address 2606:50c0:8001::153 is a 128-bit address supporting modern dual-stack networking. DNS returns A records for IPv4 and AAAA records for IPv6.

3. HTTP vs HTTPS

This site is served exclusively over HTTPS on port 443, confirmed by the browser lock icon and Chrome DevTools showing Remote Address: 185.199.110.153:443. The status code returned is 304 Not Modified, meaning the browser cache was validated. TLS encrypts all communication between your browser and GitHub's servers.
