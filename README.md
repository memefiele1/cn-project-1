## Project Overview
This project demonstrates how core networking concepts, DNS, IP addressing, HTTP/HTTPS, and security — are applied in a real-world web deployment.

## Technologies Used
- HTML & CSS
- GitHub Pages (hosting)
- HTTPS via Let's Encrypt (TLS)

## Networking Concepts Demonstrated
- **DNS:** Domain resolves via A (IPv4) and AAAA (IPv6) records
- **IP Addressing:** IPv4 (185.199.x.153) and IPv6 (2606:50c0::153)
- **HTTPS:** Enforced on port 443 using TLS over TCP/IP
- **DoD Model:** HTTPS at Application layer, TCP at Transport, IP at Internet layer
